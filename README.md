Robokassa API
=====================

Простой гем для работы с API Robokassa. Позволяет быстро организовать на своём Rails приложении (сервисе или интернет-магазине) приём онлайн-платежей.

### Установка

Добавьте в своем Gemfile

```
gem 'robokassa_api'
```
